## 详细兼容性信息

### Chrome

- **支持版本**: 1
- **部分实现**: 是
- **注意事项**:
  - The `onfocusin` event handler property is not supported. To listen to this event, use `element.addEventListener('focusin', function() {});`.

### Edge

- **支持版本**: 12
