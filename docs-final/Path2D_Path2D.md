- **支持版本**: 36

### Edge

- **支持版本**: 14
- **注意事项**:
  - Before Edge 79, the constructor for `Path2D` objects does not support invocation with a string consisting of SVG path data.

### Firefox

- **支持版本**: 31

### Safari

- **支持版本**: 8
