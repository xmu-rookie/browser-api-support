### Chrome

- **支持版本**: 61

### Firefox

- **支持版本**: 33
- **注意事项**:
  - Firefox 69 introduced support for the modern six-parameter syntax for `scaleSelf()`. Previously, it only supported the older three-parameter syntax: `scale(scaleX[, originX][, originY]]])`.

### Safari

- **支持版本**: 11
