# Document.writeln.enforces_trusted_types API 兼容性数据

## 基本信息

- **API名称**: `Document.writeln.enforces_trusted_types`
- **规范文档**: [查看规范](https://html.spec.whatwg.org/multipage/dynamic-markup-insertion.html#dom-document-writeln)
- **标签**: `web-features:document-write`, `web-features:trusted-types`
- **描述**: Requires `TrustedHTML` instance when trusted types are enforced
