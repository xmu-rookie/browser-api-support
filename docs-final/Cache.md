## 详细兼容性信息

### Chrome

- **支持版本**: 40
- **注意事项**:
  - Before version 43, only service workers are supported. From version 43, all worker types and the main thread are supported.

### Deno

- **支持版本**: 1.26

### Edge

- **支持版本**: 16

- **支持版本**: 41
