## 详细兼容性信息

### Chrome

- **支持版本**: 74
- **注意事项**:
  - On Windows and ChromeOS, the entire system audio can be captured when sharing an entire screen. On Linux and macOS, only the audio of a tab can be captured.

- **支持版本**: 不支持
