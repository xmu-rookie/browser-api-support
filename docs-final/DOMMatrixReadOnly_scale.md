### Chrome

### Firefox

- **支持版本**: 33
- **注意事项**:
  - Firefox 69 introduced support for the modern six-parameter syntax for `scale()`. Previously, it only supported the older three-parameter syntax: `scale(scaleX[, originX][, originY]]])`.

- **支持版本**: 11
