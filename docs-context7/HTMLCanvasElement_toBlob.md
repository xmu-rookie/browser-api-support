## 相关子API

该API包含以下子功能：

- **quality_parameter**: `quality` parameter
- **type_parameter_jpeg**: `type` parameter supports `"image/jpeg"`
- **type_parameter_png**: `type` parameter supports `"image/png"`
- **type_parameter_webp**: `type` parameter supports `"image/webp"`
