## 相关子API

该API包含以下子功能：

- **accepts_nodes_in_shadow_trees**: Accepts `node` parameter in any tree/shadow tree
- **node_parameter_nullable**: `node` parameter is nullable
- **offset_parameter_optional**: `offset` parameter is optional
